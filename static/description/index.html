🚚 Bon de Livraison Algérie

Solution professionnelle pour la gestion des bons de livraison conformes au context du marché algérien

📋 Pourquoi ce module ?

En Algérie, le bon de livraison est un document réglementaire obligatoire qui doit accompagner toute marchandise livrée. Notre module CPSS vous permet de générer des BL professionnels, conformes à la législation, avec une numérotation automatique et un suivi complet.

🎯 Fonctionnalités

🔢

Numérotation Automatique

Génération automatique des numéros BL au format BL/YYYY/XXXXX conforme aux standards algériens.

💰

Calculs Précis

Montants calculés automatiquement sur les quantités réellement livrées, pas commandées.

📊

Suivi Complet

États d'impression, facturation et paiement pour un suivi optimal de vos livraisons.

🔗

Intégration Parfaite

Intégré nativement avec les commandes de vente et les livraisons Odoo.

📄

PDF Professionnel

Rapport PDF conforme à la réglementation avec zones de signature obligatoires.

🎛️

Filtres Avancés

Recherche par état : BL non facturés, non payés, imprimés, etc.

⚙️ Comment ça marche ?

1

📝 Commande de Vente

Créez votre commande de vente normalement dans Odoo. Le module s'intègre parfaitement avec le processus existant.

2

📦 Préparation Livraison

Confirmez les quantités à livrer dans le picking. Le module calculera automatiquement les montants basés sur les quantités réelles.

3

🚚 Génération BL

Cliquez sur "Générer BL" pour créer automatiquement le numéro et marquer le document comme imprimé.

4

🖨️ Impression PDF

Imprimez le bon de livraison professionnel conforme au context du marché algérien avec toutes les informations requises.

5

📊 Suivi et Rapports

Suivez vos BL via le menu dédié avec des filtres avancés pour identifier les documents non facturés, non payés, etc.
